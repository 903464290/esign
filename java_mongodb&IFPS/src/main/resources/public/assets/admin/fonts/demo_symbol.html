symbol reference

This is a brand-new way of use, and it should be said that this is the mainstream in the future, and it is also the current recommended usage of the platform. Related introduction can refer to thisarticle This usage is actually a collection of svg, which has the following characteristics compared with the other two：

  • Support multi-color icons, no longer limited by single color。
  • Through some skills, support like fonts, throughfont-size,colorTo adjust the style。
  • Poor compatibility, support ie9+, and modern browsers。
  • The performance of browser rendering svg is average, not as good as png。

The use steps are as follows：

Step 1: Import the symbol code generated under the project：

<script src="./iconfont.js"></script>

Step 2: Add general css code (introduce it once）：

<style type="text/css">
.icon {
   width: 1em; height: 1em;
   vertical-align: -0.15em;
   fill: currentColor;
   overflow: hidden;
}
</style>

The third step: select the corresponding icon and get the class name, apply it to the page：

<svg class="icon" aria-hidden="true">
  <use xlink:href="#icon-xxx"></use>
</svg>
